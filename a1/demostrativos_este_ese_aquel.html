Los demostrativos

指示形容詞：este / ese / aquel

¿Para qué sirven?

話し手・聞き手からの距離で使い分けます。

Los demostrativos indican distancia entre el hablante, el oyente y el objeto.

Tabla general

Distancia Masculino Femenino Plural masc. Plural fem. 日本語
Cerca del hablante este esta estos estas これ / この
Cerca del oyente ese esa esos esas それ / その
Lejos de ambos aquel aquella aquellos aquellas あれ / あの

Ejemplos

例文と日本語訳です。

Este libro es mío. (cerca del hablante)
日本語を見る / 隠す この本は私のです。

Esa mesa es tuya. (cerca del oyente)
日本語を見る / 隠す そのテーブルはあなたのです。

Aquella casa es muy vieja. (lejos de ambos)
日本語を見る / 隠す あの家はとても古いです。

Ejercicios de repaso

1. Completa con el demostrativo correcto

正しい指示形容詞を入れましょう。

a) ___ libro (cerca de mí) Ver respuesta este libro

b) ___ zapatos (cerca de ti) Ver respuesta esos zapatos

c) ___ montaña (muy lejos) Ver respuesta aquella montaña

2. Traduce al español

スペイン語に訳しましょう。

a) この犬 Ver respuesta este perro

b) その人 Ver respuesta esa persona

c) あの学校 Ver respuesta aquella escuela

Mini‑quiz

正しい文を選びましょう。

1) （意味：「あの車は高いです。」）

a) Ese coche es caro.
b) Aquel coche es caro.
答えを見る 正解：b) Aquel coche es caro。

2) （意味：「この花はきれいです。」）

a) Esta flor es bonita.
b) Esa flor es bonita.
答えを見る 正解：a) Esta flor es bonita。
